# Reusable Prompts

## Start a TODO item (with planning)

Inspect the project and come up with a plan for item #___ in TODO.md.

## Start a TODO item (direct implementation)

Inspect the project and do item #___ in TODO.md. This should be a pretty small item, so proceed directly to the implementation unless you have any critical questions.

## After providing answers

Answers provided in TODO.md. Proceed with the implementation.

## Update main working directory

Update the main working directory (not the worktree for this session) with the latest changes from main.
